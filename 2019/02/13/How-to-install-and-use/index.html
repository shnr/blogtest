How to install and use

Hexo

Markdownを使うによいブログを探していた所、素晴らしいエディタを発見。

https://hexo.io/themes/

これからこれで書こうかな。

githubのページに、デプロイしてみたのでその手順を。
ちなみに30分位ですべて完了。

Githubページの作成

Githubではレポジトリのブランチをgh-pagesにすることで、ページが作成出来る。

まず普通にレポジトリを作り、ローカルにclone.

そしてディレクトリに入り、gh-pagesブランチを作成する。

試しにindex.htmlを作成してcommit.

すると、http://hoge.github.io/fuga で、
hellow worldが表示されるはず。

Hexoのインストール

こいつも超簡単。

https://hexo.io/
TOPページにある通り、

ひとまずこれで動く。

記事を書く

hexo new コマンドで.mdファイルを作成し、
その中で書くみたい。

Markdownで書いた後は、generateコマンド。

とすればhtmlやassetsが生成される。

デプロイ

先程作ったgithubページへデプロイする。
_config.yml というファイルを編集する。
下のページが詳しい。

https://hexo.io/docs/deployment#How-does-it-work-exactly

２つのgithubアカウントをつかっているので、repoのURLはhoge@というようにユーザー名を指定した。
編集後、再度generateしてデプロイする。

これで自動的にレポジトリにプッシュされ、公開されるという手軽さ。

今後はテーマやデザイン変更などもみていきたい。
